Your registry

Add your key to the registry and verify received vaccinations, or verify vaccinations others have shared with you.

As one of the goals of this registry is eliminate the problems that arise when the "Impfpass" is lost, generating a new keypair as it is done in this implementation is not possible. The private key can be lost aswell, which also results in the information being unreadable.
This can however be solved if the used keypair is derived from a secret, that only the vaccinated person can know using a key derivation function. One such secret could be a predefined value signed using the "Handy-Signatur".

To encrypt and decrypt and to generate keys this implementation uses ethereum's crypto library. Since the functionality of the smart contract does not depend on the type of encryption used it can be changed to any other system.

Only the derived public key will be saved in the registry - don't lose the private key!
